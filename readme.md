# 💈 Barbershop Manager 

Sistema de agendamento online para barbearias. 

## 🎯 Por Quê? 

Negócios baseados em agendamentos frequentemente perdem clientes por não terem um agendamento online. Resolvemos isso com uma página simples, rápida e gratuita.

## 🛠️ Tecnologias 

- **HTML5** - Estrutura 
- **CSS3** - Design responsivo 
- **JavaScript ES6+** - Interatividade 
- **Leaflet** - Mapa interativo (API)
